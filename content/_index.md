# AWS Solutions Architect Associate (SAA-C03) Study Guide

## Overview

The AWS Certified Solutions Architect - Associate (SAA-C03) exam validates your ability to design solutions based on the AWS Well-Architected Framework. This comprehensive workshop covers all four exam domains with theory, hands-on labs, mock exams, cheat sheets, flashcards, and comparison tables.

## Exam Structure

- **Format:** 65 questions (50 scored + 15 unscored)
- **Types:** Multiple choice and multiple response
- **Duration:** 130 minutes
- **Passing Score:** 720 / 1000
- **Cost:** $150 USD

## Content Domains and Weightings

| Domain | Weight | Description |
|--------|--------|-------------|
| Domain 1 | 30% | Design Secure Architectures |
| Domain 2 | 26% | Design Resilient Architectures |
| Domain 3 | 24% | Design High-Performing Architectures |
| Domain 4 | 20% | Design Cost-Optimized Architectures |

## Learning Objectives

After completing this workshop, you will be able to:

- Design secure access strategies using IAM, VPC, and encryption services
- Architect resilient, highly available solutions with proper DR strategies
- Select high-performing storage, compute, database, and network solutions
- Optimize costs across all AWS service categories
- Apply the AWS Well-Architected Framework to real-world scenarios
- Confidently answer SAA-C03 exam questions across all domains

## How to Use This Guide

Each task page in this workshop includes:

1. **Theory** — Detailed explanations of AWS services and concepts
2. **Hands-On Lab** — Step-by-step practical exercises with AWS CLI and Console
3. **Flashcards** — Quick-review question-answer pairs
4. **Mock Exam Questions** — Practice questions in exam format with explanations
5. **References** — Links to AWS documentation, blogs, and whitepapers

Work through the domains in order, or jump to specific topics you need to review. Use the mock exam section to assess your readiness.
