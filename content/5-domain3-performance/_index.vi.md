# Domain 3: Thiết kế kiến trúc hiệu suất cao

## Trọng số: 24% nội dung tính điểm

Domain này bao gồm lựa chọn giải pháp hiệu suất cao và có khả năng mở rộng cho lưu trữ, tính toán, cơ sở dữ liệu, mạng và nhập/chuyển đổi dữ liệu.

## Các Task Statement

| Task | Mô tả |
|------|-------|
| Task 3.1 | Xác định giải pháp lưu trữ hiệu suất cao |
| Task 3.2 | Thiết kế giải pháp tính toán hiệu suất cao |
| Task 3.3 | Xác định giải pháp cơ sở dữ liệu hiệu suất cao |
| Task 3.4 | Xác định kiến trúc mạng hiệu suất cao |
| Task 3.5 | Xác định giải pháp nhập và chuyển đổi dữ liệu hiệu suất cao |
