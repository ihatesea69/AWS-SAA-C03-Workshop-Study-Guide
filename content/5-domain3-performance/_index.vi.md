# Domain 3: Thiet ke kien truc hieu suat cao

## Trong so: 24% noi dung tinh diem

Domain nay bao gom lua chon giai phap hieu suat cao va co kha nang mo rong cho luu tru, tinh toan, co so du lieu, mang va nhap/chuyen doi du lieu.

## Cac Task Statement

| Task | Mo ta |
|------|-------|
| Task 3.1 | Xac dinh giai phap luu tru hieu suat cao |
| Task 3.2 | Thiet ke giai phap tinh toan hieu suat cao |
| Task 3.3 | Xac dinh giai phap co so du lieu hieu suat cao |
| Task 3.4 | Xac dinh kien truc mang hieu suat cao |
| Task 3.5 | Xac dinh giai phap nhap va chuyen doi du lieu hieu suat cao |
